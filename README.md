# IMoocMall

> 简易网上商城项目

## 项目结构

- imoocmall       旧版网上商城

- lemall          升级版网上商城

- vue-toast-demo  自己制作插件

- webpack_demo    webpack练习

## imoocmall 

```bash
cd imoocmall

npm install

npm run dev
```

## lemall 

```bash
cd lemall

npm install

npm run serve
```

## webpack_demo 

```bash
cd webpack_demo

npm install

webpack
```
